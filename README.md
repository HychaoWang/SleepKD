# Teacher Assistant-Based Knowledge Distillation Extracting Multi-level Features on Single Channel Sleep EEG (sleepkd)

This is the repo for the sleepkd project, which aims to compress the large-scale sleep staging models.

The repo contains:
- The code of sleepkd.
- The code of DeepSleepNet, SalientSleepNet and their corresponding student model.
- The sleepkd-compressed DeepSleepNet and SalientSleepNet.
- The webpage, poster of sleepkd.
